# IPTNP – Interview Preparation & Tracking for Notified Placements

IPTNP is a web-based interview preparation platform that provides company-wise and round-wise interview questions for candidates after shortlisting.

## Features
- Admin-controlled interview questions
- Company and round based filtering
- Simple and clean UI
- PHP & MySQL based application

## Tech Stack
- PHP
- MySQL
- HTML, CSS
- Apache (XAMPP)

## How to Run
1. Install XAMPP
2. Start Apache & MySQL
3. Import `iptnp_database.sql` in phpMyAdmin
4. Place project in `htdocs`
5. Open `http://localhost/IPTNP`
